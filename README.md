# CGRetry
An ULTRAKILL mod that allows you to **retry the current wave mid-game or upon death** in the **Cyber Grind**. Useful for **training** on difficult waves.

# Usage
You can retry a wave either:
* From the pause menu (replaces the checkpoint button).
* From the dialog that appears upon death. 

> Retrying will enable cheats so your score won't be saved.

> Retrying is disabled when the Cyber Grind hasn't started, or during wave transitions.

# Configuration
You can tweak values such as the **startup time** and **start rank** for retried waves to better fit your needs.
